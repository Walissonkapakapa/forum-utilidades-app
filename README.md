# Fórum de Utilidades & Métodos

Fórum simples e bonito com sistema de **perfil pessoal** (nickname + foto), publicações, curtidas e comentários.

## Como usar

1. Abra o `index.html` no navegador
2. Clique em **Entrar** (canto superior direito)
3. Escolha seu apelido e foto de perfil (ou envie uma sua)
4. Publique métodos úteis, scripts e dicas
5. Curta e responda outros posts

Tudo fica salvo no seu navegador (localStorage).

## Hospedar no GitHub Pages (gratuito)

1. No repositório, vá em **Settings > Pages**
2. Source: **Deploy from a branch** → `main` / `/ (root)`
3. Save
4. Aguarde e acesse o link gerado

Seu fórum vai estar online e acessível pelo celular!

**Regras da comunidade:** Apenas conteúdo útil e legítimo. Sem burla de sistemas ou fraudes.

Feito com HTML + Tailwind + JavaScript.
